# ICARO REACT

## Taller 1

### Requisitos

Este repo fue creado con Vite y requiere

```
node v16.0.0
yarn o npm
```

### Instalación

yarn:

```bash
git clone https://github.com/aalvarezto/icaro-react-taller-1.git
cd icaro-react-taller-1
yarn
```

npm:

```bash
git clone https://github.com/aalvarezto/icaro-react-taller-1.git
cd icaro-react-taller-1
npm install
```
